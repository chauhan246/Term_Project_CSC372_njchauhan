Import Products in Bulk

File Format Requirements

Please upload your data in one of the following formats:

  • JSON (.json)
  • CSV (.csv)
  • TXT (.txt)

JSON Format example:

{
  "products": [
    {
      "prod_id": "1",
      "prod_name": "Luxury fragrance product 1",
      "prod_image": "path/to/image.jpg",
      "prod_desc": "product 1 has a floral like scent.",
      "prod_category": "male",
      "prod_price": 49.00
    }
  ]
}

CSV Format example:

 prod_id,prod_name,prod_image,prod_desc,prod_category,prod_price
 "1","Luxury fragrance product 1","path/to/image.jpg","product 1 has a floral like scent.","male",49.00

TXT Format example:

1|Luxury fragrance product 1| path/to/image.jpg | product 1 has a floral like scent.| male | 49.00
